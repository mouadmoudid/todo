# todo

A new Flutter project.

## Nouvelle fonctionnalités ajoutées

Ces améliorations ajoutent support de:

- Catégories / Projets (création, suppression, couleur personnalisée)
- Tags (ex: #urgent) et filtrage par tag
- Sous-tâches avec barre de progression automatique

## Comment tester

1. Lancer l'app:

```powershell
cd 'c:\Users\HP\Desktop\flutter\tp3.1\todo-APP\todo'
flutter run
```

2. Se connecter, puis sur la page principale :
	- Ajouter une tâche via « Ajouter » — cliquez sur le bouton pour ouvrir la fenêtre de création complète (ajout de tags / sous-tâches / catégories).
	- Gérer les catégories via l'icône dossier dans la barre d'application (créer une nouvelle catégorie, choisir une couleur).
	- Filtrer par catégorie et par tag depuis la zone de filtre sous l'ajout de tâche.
	- Ouvrir une tâche pour voir les sous-tâches et la barre de progression; cocher les sous-tâches mettra à jour l'état parent automatiquement.
